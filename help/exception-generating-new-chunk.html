Minecraft Crash Help

Exception generating new chunk

Errors while loading certain chunks can be caused by a wide variety of issues.

In order to help narrow the issue down, see the following cases below.

Error: Invalid Biome ID

Note: This issue will be resolved in future version 1.8 and is resolved in 14w02c.

If you are encountering this error, please try the following:

  • Creating a new world specifying a valid http://minecraft.gamepedia.com/Biome#Biome_IDs.
  • Loading a New World, without customization settings.
  • Reinstall Minecraft and or server, to remove the possibility of external modifications

Mojira issue: MC-44380

Error: Exception ticking world

Note: This issue will be resolved in future version 1.8 and is resolved in 14w02c and 14w03a.

This error can be caused by having chunks that are 'corrupted' and require repair.

You can, however try the following to manually repair your world:

  • Run MCEdit*, a map editing tool which can be used for chunk repair.
  • Run a community made 'Minecraft Region Fixer' tool for repairing corrupt chunks.
  • Ask one of the community support references for further information and or help (please be sure to link back to here!).

Mojira issue: MC-44824
