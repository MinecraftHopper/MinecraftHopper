Incompatible class change error

These errors are very vague, there's no universal fix.

Common causes are client or server modifications, corrupt world data, and or a bug in the game.

General overview

Many of the issues directed to this error, are caused by the following:

  1. Client or Server modifications
  2. Corrupt world data
  3. In some cases, a bug with the actual game

Please contact our technical support resources for more information, when contacting them please link back to this page.
